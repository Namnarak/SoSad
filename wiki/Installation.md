# Installation

## Requirements

- Python 3.12+
- hikari >= 2.0.0
- pydantic >= 2.0

## Install with pip

```bash
pip install sosad
```

## Install with uv (recommended)

```bash
uv add sosad
```

## Install with poetry

```bash
poetry add sosad
```

## Install with pipenv

```bash
pipenv install sosad
```

## Development install

```bash
git clone https://github.com/Namnarak/SoSad.git
cd SoSad
uv sync --dev
```

## Verify installation

```bash
python -c "import sosad; print(sosad.__version__)"
# 0.1.2
```
