# Installation

## Requirements

- Python 3.10+
- A Discord Bot Token ([Discord Developer Portal](https://discord.com/developers/applications))

## Install with pip

```bash
pip install sosad
```

## Install with uv

```bash
uv add sosad
```

## Install from source

```bash
git clone https://github.com/Namnarak/SoSad.git
cd SoSad
uv sync
```

## Verify installation

```bash
python -c "import sosad; print(sosad.__version__)"
```
