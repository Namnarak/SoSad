# Installation

## Requirements

- Python 3.12+
- hikari >= 2.0.0
- pydantic >= 2.0

## Install

=== "pip"

    ```bash
    pip install sosad
    ```

=== "uv"

    ```bash
    uv add sosad
    ```

=== "poetry"

    ```bash
    poetry add sosad
    ```

## Verify

```bash
python -c "import sosad; print(sosad.__version__)"
```
